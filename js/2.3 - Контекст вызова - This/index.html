2.3 - Контекст вызова - This

Контекст вызова - This

Контекст вызова (this) - это объект, в контексте которого была вызвана функция. Влияет на this и переданные аргументы внутрь функции.

Контекст выполнения и контекст вызова - это разные понятия, и важно понимать, что они относятся к разным аспектам выполнения кода в JavaScript.

Контекст это тот кто вызвал функцию, тот для кого она выполняется.

Контекст (контекст вызова) - это объект, который "заново определяется" при КАЖДОМ вызове функции. Доступ к контексту осуществляется через переменную This. Эту переменную не нужно создавать отдельно, она уже существует, если функция вызвана.

Краткий итог:

1. В глобальной области видимости this - window (по сути это всегда, если она вызвана не как метод)

2. При вызове метода объекта this укажет на этот самый объект

3. Внутри слушателя события this указывает на объект, на котором висит слушатель (по сути то же самое что event.currentTarget)

4. Если функция вызвана как конструктор, то this становится вновь созданный объект (не проходили)
